Returns & Exchanges

If something isn’t right, we make returns simple and clear.

Return Window

Returns can be requested within 14 days of delivery. Items must be unworn and returned in their original packaging.

How to Request a Return

  1. Contact us via the About page.
  2. Include your order number and reason for return.
  3. We will confirm the return instructions.

Exchanges

Exchanges depend on stock availability. If an exchange is not possible, a refund can be arranged.
